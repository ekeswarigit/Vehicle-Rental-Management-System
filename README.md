##he Vehicle Rental Management System is a database-driven application designed to simplify the process of renting vehicles. 
The system enables users to add vehicles, register customers, and make bookings efficiently using a structured backend. 
This project is implemented using Hibernate ORM for seamless interaction with the MySQL database, ensuring automated mapping between Java objects and database tables. 
It eliminates manual queries, reduces coding complexity, and enhances performance with reliable transaction management.
